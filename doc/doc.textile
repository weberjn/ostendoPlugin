h1(#install). Installation

Copy the plugin jar to eclipse/plugins. 
Restart Eclipse.


h1(#usage). Usage

# add one or more IDL files
# add one or more log directories

# Select a request message (right arrow icon), and optionally the corresponding reply message
# If the operation name is unique in the added IDLs, the operation is selected in the IDL tree. If it is not unique the user has to select the operation.
# Klick Run
