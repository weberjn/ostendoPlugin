Installation

Copy the plugin jar to eclipse/plugins.
Restart Eclipse.

Usage

  1. add one or more IDL files
  2. add one or more log directories
  1. Select a request message (right arrow icon), and optionally the corresponding reply message
  2. If the operation name is unique in the added IDLs, the operation is selected in the IDL tree. If it is not unique the user has to select the operation.
  3. Klick Run
